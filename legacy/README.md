# Legacy

Este diretório guarda o protótipo antigo que já funcionava como referência de implementação.

## Conteúdo preservado

- `node-express-prototype/server.js`
- `node-express-prototype/public/`
- `node-express-prototype/package.json`
- `node-express-prototype/package-lock.json`
- `node-express-prototype/discover.js`

## Por que ficou aqui

Esse protótipo é útil como base para as próximas etapas de migração de:

- FFmpeg
- MJPEG
- RTSP
- ONVIF PTZ

Nada aqui deve ser tratado como a arquitetura final. É apenas referência funcional.
